# Optimization Portfolio

This repository contains a collection of algorithms and homework assignments focused on mathematical optimization, routing problems, and portfolio management.

## Project Structure

* **`/Data/`**: Shared data folder containing `races.csv` and `circuits.csv` used across multiple assignments.
* **`/HW1_Routing/`**: Implementation of routing algorithms and heuristic search.
* **`/HW2_TSP_Dates/`**: Advanced Traveling Salesman Problem (TSP) implementation featuring:
    * **Jupyter Notebook**: Full analysis and algorithmic approach.
    * **`tsp_solution_map_with_dates.html`**: An interactive map visualization of the TSP route.
    * **`calendar.jpg`**: Supporting temporal data for the optimization logic.
* **`/Portfolio_Optimization/`**: Independent study on Risk/Return portfolio simulations, utilizing efficient frontier optimization techniques.

## Algorithms Explored
- **Traveling Salesman Problem (TSP)**: Solving for optimal routes with temporal constraints.
- **Routing Heuristics**: Implementing and comparing search strategies.
- **Portfolio Optimization**: Balancing risk and return in financial simulations.

## Visualization
The project includes interactive HTML maps to better understand the spatial-temporal results of the optimization algorithms.
